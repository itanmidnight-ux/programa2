1.1 RSI INCORRECTO (signal-engine.ts, analysis-engine.ts)
------------------------------------------------------------------------
   PROBLEMA: Usaba promedio simple (SMA) en lugar del método de Wilder
   UBICACIÓN: signal-engine.ts:35-48, analysis-engine.ts:185-214
   
   ANTES (INCORRECTO):
   ```typescript
   const avgGain = gains.reduce((a, b) => a + b, 0) / period;
   const avgLoss = losses.reduce((a, b) => a + b, 0) / period;
   ```
   
   DESPUÉS (CORRECTO - Wilder Smoothing):
   ```typescript
   // Inicialización con SMA
   for (let i = 1; i <= period; i++) { ... }
   avgGain /= period;
   avgLoss /= period;
   
   // Wilder's smoothing: (prev * (period-1) + current) / period
   avgGain = (avgGain * (period - 1) + gain) / period;
   avgLoss = (avgLoss * (period - 1) + loss) / period;
   ```

1.2 EMA MAL INICIALIZADA (signal-engine.ts, analysis-engine.ts)
------------------------------------------------------------------------
   PROBLEMA: Usaba data[0] como valor inicial en lugar de SMA del período
   UBICACIÓN: signal-engine.ts:25-33, analysis-engine.ts:110-124
   
   ANTES (INCORRECTO):
   ```typescript
   result[0] = data[0];
   for (let i = 1; i < data.length; i++) {
     result[i] = (data[i] - result[i - 1]) * multiplier + result[i - 1];
   }
   ```
   
   DESPUÉS (CORRECTO - SMA Initialization):
   ```typescript
   // Inicializar con SMA del período
   let sma = 0;
   for (let i = 0; i < period; i++) sma += data[i];
   sma /= period;
   
   // Llenar primeros 'period' valores con SMA
   for (let i = 0; i < period; i++) result.push(sma);
   
   // Continuar con EMA estándar
   let prevEMA = sma;
   for (let i = period; i < n; i++) {
     const ema = data[i] * k + prevEMA * (1 - k);
     result.push(ema);
     prevEMA = ema;
   }
   ```

1.3 PnL IRREAL (execution-engine.ts)
------------------------------------------------------------------------
   PROBLEMA: No incluía fees, slippage ni costos reales por trade
   UBICACIÓN: execution-engine.ts:696-827
   
   ANTES: Solo calculaba gross PnL sin costos
   DESPUÉS: Nueva función calculateRealisticPnL()
   ```typescript
   export function calculateRealisticPnL(
     entryPrice, exitPrice, quantity, side,
     entryFeeRate = TAKER_FEE,
     exitFeeRate = TAKER_FEE,
     slippageRate = SLIPPAGE
   ) {
     // Gross PnL
     const grossPnl = side === 'LONG' 
       ? (exitPrice - entryPrice) * quantity
       : (entryPrice - exitPrice) * quantity;
     
     // Fees (entrada + salida)
     const entryFee = entryPrice * quantity * entryFeeRate;
     const exitFee = exitPrice * quantity * exitFeeRate;
     const fees = entryFee + exitFee;
     
     // Slippage
     const slippage = entryPrice * quantity * slippageRate;
     
     // Net PnL
     const netPnl = grossPnl - fees - slippage;
     return { grossPnl, fees, slippage, netPnl, netPnlPercent };
   }
   ```
